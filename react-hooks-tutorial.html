React Hooks: A Modern Fejlesztés Alapjai

📅 2025. Jan 12. ⏰ 8 perc olvasás 👤 Roland

🚀 Bevezetés a React Hooks világába

A React Hooks forradalmasították a modern React fejlesztést. A React 16.8 verziójában bevezetett Hooks lehetővé teszik, hogy funkcionális komponensekben is használhassuk a state-et és más React funkciókat, anélkül hogy class komponenseket kellene írnunk.

🎯 Mit fogsz megtanulni ebben a cikkben:
  • useState és useEffect Hook-ok alapjai
  • Custom Hook-ok készítése
  • Performance optimalizálási technikák
  • Gyakori hibák elkerülése
  • Best practice-ek és pro tippek

1. useState Hook - State Kezelés Funkcionális Komponensekben

A useState a leggyakrabban használt Hook, amely lehetővé teszi a state kezelését funkcionális komponensekben. Egyszerű és intuitív használata miatt ideális kezdőknek és haladóknak egyaránt.

Alapvető használat:

import React, { useState } from 'react'; function Counter() { const [count, setCount] = useState(0); const [user, setUser] = useState(null); const [loading, setLoading] = useState(false); return ( <div> <p>Számláló: {count}</p> <button onClick={() => setCount(count + 1)}> Növelés </button> </div> ); }

2. useEffect Hook - Side Effect-ek Kezelése

Az useEffect Hook segítségével side effect-eket kezelhetünk, mint például API hívások, DOM manipuláció, vagy feliratkozások. Ez helyettesíti a class komponensek componentDidMount, componentDidUpdate és componentWillUnmount lifecycle metódusait.

Példa API hívásra:

import React, { useState, useEffect } from 'react'; function UserProfile({ userId }) { const [user, setUser] = useState(null); const [loading, setLoading] = useState(true); useEffect(() => { const fetchUser = async () => { setLoading(true); try { const response = await fetch(`/api/users/${userId}`); const userData = await response.json(); setUser(userData); } catch (error) { console.error('Hiba a felhasználó betöltésekor:', error); } finally { setLoading(false); } }; fetchUser(); }, [userId]); // Dependency array - csak userId változásakor fut újra if (loading) return <div>Betöltés...</div>; if (!user) return <div>Felhasználó nem található</div>; return <div>Üdv, {user.name}!</div>; }

3. Custom Hook-ok - Logika Újrafelhasználása

A Custom Hook-ok lehetővé teszik, hogy a komponensek közötti logikát újrafelhasználható módon szervezzük ki. Egy custom hook egyszerűen egy JavaScript függvény, amely más Hook-okat használ.

useApi Custom Hook példa:

import { useState, useEffect } from 'react'; function useApi(url) { const [data, setData] = useState(null); const [loading, setLoading] = useState(true); const [error, setError] = useState(null); useEffect(() => { const fetchData = async () => { try { setLoading(true); setError(null); const response = await fetch(url); if (!response.ok) { throw new Error(`HTTP error! status: ${response.status}`); } const result = await response.json(); setData(result); } catch (err) { setError(err.message); } finally { setLoading(false); } }; fetchData(); }, [url]); return { data, loading, error }; } // Használat: function ProductList() { const { data: products, loading, error } = useApi('/api/products'); if (loading) return <div>Betöltés...</div>; if (error) return <div>Hiba: {error}</div>; return ( <ul> {products.map(product => ( <li key={product.id}>{product.name}</li> ))} </ul> ); }

4. Performance Optimalizálás

A React Hook-ok használatakor fontos figyelni a performance-ra. Itt néhány hasznos technika:

useMemo és useCallback:

import React, { useMemo, useCallback } from 'react'; function ExpensiveComponent({ items, onItemClick }) { // Költséges számítás memoizálása const expensiveValue = useMemo(() => { return items.reduce((sum, item) => sum + item.value, 0); }, [items]); // Callback függvény memoizálása const handleClick = useCallback((id) => { onItemClick(id); }, [onItemClick]); return ( <div> <p>Összeg: {expensiveValue}</p> {items.map(item => ( <button key={item.id} onClick={() => handleClick(item.id)}> {item.name} </button> ))} </div> ); }

💡 Pro Tippek és Best Practice-ek:

  • Dependency Array: Mindig használj dependency array-t useEffect-nél, hogy elkerüld a végtelen loop-okat
  • State Struktura: Kerüld a túl sok useState használatát - használj useReducer-t komplex state-nél
  • Custom Hook-ok: Szervezd ki az újrafelhasználható logikát custom hook-okba
  • Cleanup: Ne felejtsd el a cleanup függvényeket useEffect-ben (pl. event listener-ek eltávolítása)
  • React DevTools: Használd a React DevTools-t a Hook-ok debug-olásához
  • ESLint Plugin: Telepítsd az eslint-plugin-react-hooks-ot a Hook szabályok betartásához

5. Gyakori Hibák és Megoldásaik

❌ Helytelen dependency array:

// Rossz - végtelen loop useEffect(() => { setCount(count + 1); }, [count]); // Jó - functional update használata useEffect(() => { setCount(prevCount => prevCount + 1); }, []);

❌ Missing cleanup:

// Rossz - memory leak useEffect(() => { const interval = setInterval(() => { console.log('Timer tick'); }, 1000); }, []); // Jó - cleanup függvénnyel useEffect(() => { const interval = setInterval(() => { console.log('Timer tick'); }, 1000); return () => clearInterval(interval); }, []);
